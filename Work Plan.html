üéØ PROJECT OVERVIEW
The Red River DTN & PLC Upgrade project involves comprehensive modernization of the terminal control system infrastructure. This includes installation of new network infrastructure, PLC panels, and terminal kiosks across the North Terminal (6 kiosks), South Terminal (2 kiosks), and South Shop/MCC (radio equipment). The project will establish a robust communication network using fiber optic uplinks and CAT6 Ethernet connections, with centralized control through upgraded PLC panels. All installations will follow industry standards for industrial automation and include comprehensive testing and documentation prior to cutover.
üìê SCOPE OF WORK
North Terminal: Install junction box on North Pipe Rack with network switch and fiber termination panel. Run 7 power cables (24 VDC) and 7 Ethernet cables (CAT6) serving 6 kiosks plus switch infrastructure. Install ~400' of conduit including vertical and horizontal runs.

South Terminal: Replace existing PLC panel in office building. Install 2 power cables and 2 CAT6 Ethernet cables serving 2 kiosks. Install ~240' of conduit along loading bays with vertical/horizontal drops.

South Shop/MCC: Install radio equipment with power and network connectivity (lengths TBD pending site survey).

Testing & Documentation: Perform continuity and connectivity testing for all CAT6 runs. Complete point-to-point testing for all terminations. Provide comprehensive test documentation and as-built drawings.

Coordination: I&E Department handles conduit installation and cable pulling. Automation Department provides drawings, equipment procurement, fiber terminations, and final documentation. All work requires Engineering approval before installation.
